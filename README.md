# Accounts-Rental-Frontend
This repo is all about the Accounts Rental Frontend - Where we can place an AD of renting our netflix, instagram, youtube accounts for rental for hours, days, months, years and get paid for sharing the account with them.

# Accounts-Rental Frontend (React + TypeScript)

This is a minimal Vite + React + TypeScript starter scaffold.

Getting started

1. Install dependencies

```bash
cd /home/nextpage/Desktop/AccountsFrontend
npm install
```

2. Run dev server

```bash
npm run dev
```

3. Build for production

```bash
npm run build
npm run preview
```

4. Type-check

```bash
npm run type-check
```

Files of interest

- `package.json` — scripts and dependencies
- `vite.config.ts` — Vite configuration
- `tsconfig.json` — TypeScript configuration
- `src/main.tsx` — app entry
- `src/App.tsx` — example component
